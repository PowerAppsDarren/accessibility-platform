# UTMB Accessibility Platform

## Overview
Full-stack web application for tracking UTMB IT assets, personnel, and accessibility compliance across departments, websites, and applications.

## Tech Stack
- **Frontend**: React + TypeScript + Vite, shadcn/ui (Radix), Tailwind CSS, Framer Motion
- **API**: tRPC with React Query
- **Backend**: Express + tRPC
- **Database**: MySQL + Drizzle ORM
- **Auth**: OAuth (jose JWT)
- **Testing**: Vitest

## Current Milestone
Complete the database migration: wire all frontend pages to tRPC backend, remove mock data layer, verify with tests.
